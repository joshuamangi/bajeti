# Functional Requirements [User interests]

## Users

- [x] Users should be able to register themselves in the system
- [x] Users should be able to login to the system
- [x] Users should be able to logout of the system
- [x] Users should be able to have a Remember Me option
- [x] Users should be able to view their profile
- [x] Users should be able to edit their profile

## Budget

- [ ] Users should be able to create a budget
- [ ] Users should be able to have the ability to add multiple budgets
- [ ] Users should be able to select from different budget templates
- [ ] Users should be able to edit a Budget
- [ ] Users should be able to delete a Budget
- [ ] Users should be able to share a Budget

## Categories

- [x] Users should be able to create Categories
- [x] Users should be able to view Categories
- [x] Users should be able to edit Categories
- [x] Users should be able to delete Categories
- [ ] Users should be able to filter Categories with a slider
- [ ] Users should be able to view Category Limits status
- [ ] Users should be able to flag if a Category is recurring monthly

## Expenses

- [x] Users should be able to create Expenses per Category
- [x] Users should be able to view Expenses per Category
- [x] Users should be able to edit Expenses per Category
- [x] Users should be able to delete Expenses per Category

## Transfers

- [x] Users should be able to transfer funds from one category to the other
- [x] Users should be able to undo transfers done from or to a category
- [x] Users should be able to view transfers done on a category

## Data

- [ ] Users should be able to generate their data based on criteria
- [ ] Users should be able to export their data
- [ ] Users should be able to remove their data
- [ ] Users should be able to archive their data
